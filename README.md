# 📄 IARE Worksheet Portal

A clean, single-page web app that lets IARE students instantly access and view their lab worksheet PDFs stored on the college's AWS S3 bucket.

## ✨ Features

- 🔍 Load any worksheet by entering Roll Number, Semester, Course Code, and Week
- 📥 Download PDFs directly from S3
- 🕒 Recent access history (stored in localStorage)
- 🌙 Dark / ☀️ Light mode toggle
- 📱 Fully responsive design
- 🎨 Glassmorphism UI with smooth animations

## 🛠️ Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- [Bootstrap 5.3](https://getbootstrap.com/)
- [Google Fonts](https://fonts.google.com/) — Bebas Neue & Work Sans
- AWS S3 (for PDF storage, managed by IARE)

## ⚠️ Disclaimer

This tool uses publicly accessible S3 URLs provided by IARE. It does not store, modify, or redistribute any student data.
